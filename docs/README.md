# Documentación NovaMarket

La documentación oficial del proyecto está en esta carpeta.

**Empezar aquí:** [index.md](index.md)

## Generar sitio (MkDocs)

```bash
pip install mkdocs-material
mkdocs serve
```

Abrir http://localhost:8000
